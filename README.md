# ESP32-S3 Storybook Console

A standalone ESP-IDF and ESP-Brookesia project for the Waveshare ESP32-S3-Touch-AMOLED-2.06.

This project started from the merged official `01-06` demos and evolves them into a child-friendly launcher with app-based features.

Current apps:

- `Story TV`
- `Music Colors`
- `Rocket Tilt`
- `Toy Box`
- `Parent Care`
- `Device Care`

Project notes:

- Built with ESP-IDF 5.5.x
- Target board: Waveshare ESP32-S3-Touch-AMOLED-2.06
- Uses Brookesia as the host shell and each feature as an app component
